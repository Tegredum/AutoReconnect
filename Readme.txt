程序应用场景：
比如说，您在实验室里有一个台式机，平时连的是ZJUWLAN无线网。
您有些时候去不了实验室，但需要用实验室电脑里的软件或是其他文件。
您发现了TeamViewer，于是平时去不了实验室的时候就用TeamViewer连实验室的电脑。
（谁让校网无线网的IP地址不固定呢……）

然而，校网每隔14天就会自动把你在实验室的账号踢下线，然后您就不得不亲自去一次实验室，重新登录账号，联网。
校网账号没钱了，账号也会自动下线，充完钱之后也要手动登录。
然而有些时候您甚至不在杭州……
所以您有两个选择：
1. 呼叫同实验室的师兄师弟；
2. 写一个能够自动执行重连校网操作的脚本。

我选择了2。（不想麻烦别人是这样的）

程序运行环境要求：
系统：Windows（作者用的Win11x64）
运行语言：Python 3（作者用的Python 3.10.7 x64）
Python 第三方库依赖：
	1. Selenium（安装方法：pip install selenium）
	2. subprocess（pip install subprocess，可能不用装？记不清了）
	3. json（pip install json，您要把您的信息写到 JSON 文件里面，为了解析 JSON 文件，需要这个库）
其他软件依赖：
	Chrome浏览器、Chromedriver
	（配置方法几句说不清，我留个链接：https://blog.csdn.net/zhoukeguai/article/details/113247342）

-------------------------------------------------分割线-------------------------------------------------------

使用方法：
1. 打开credential.json，把您的学号和校网账号对应的密码填进相应的位置。
2. 保存credential.json，然后把AutoReconnect.py与credential.json放到同一目录下，运行AutoReconnect.py。
3. 挂着。

-------------------------------------------------分割线-------------------------------------------------------

注意事项：
1. 电脑的 WLAN 功能必须处于开启状态。因为本人比较懒，做出自己能用的功能之后就不做别的了，就没加判断 WLAN 是否打开的部分，如果 WLAN 没开，实测会报错。
2. 本人比较懒，没有给 JSON 文件加密。信不过程序安全性的话，慎用。
3. 可能存在一些我本人也没发现的 BUG。我发现之后会修，测试通过之后，这边也会更新（忙的话可能会慢一段时间）
